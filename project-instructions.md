Необходимо разработать систему, реализующую улучшение изображений в реальном времени посредством запуска ML-модели в браузере пользователя. Улучшение изображений производится по следующим параметрам: яркость, контрастность и цветность.

## Требования

— Работоспособность во всех массовых современных браузерах
— Суммарный объём кода — до 10 МБ
— Обработка изображений до 15 Мпк
— Максимальное время обработки — до 30 с
— Среднее время обработки изображения — 5 с
— Поддержка форматов: JPG, PNG, HEIC, BMP
— Работа в асинхронном режиме (без блокировки браузера пользователя)

## Порядок работы реализованной системы

1. Получение исходного изображения посредством API
2. Обработка изображения с информированием о ходе процесса (события API либо метод API — получения статуса)
3. Выгрузка готового изображения

## Рекомендуемые API модуля

### Методы

— Метод постановки задачи — получает исходное изображение в удобном виде на обработку, возвращает идентификатор задачи
— Метод получения статуса задачи — получает идентификатор задачи, возвращает текущий статус и прогресс выполнения
— Метод прерывания задачи — получает идентификатор задачи, возвращает информацию об успешном выполнении
— Метод получения готового изображения — получает идентификатор задачи, возвращает готовое изображение

### События

— Событие изменения статуса задачи — содержит идентификатор задачи, текущий статус и прогресс выполнения. Возникает при изменении статуса или прогресса выполнения задачи

## Какой результат хотим получить

- ИИ подбирает оптимальные параметры коррекции яркости, контрастности и цветности. Вспомогательный алгоритм применяет эти параметры к изображениям
- Решение успешно работает для всех эталонных изображений
- Решение работает во всех современных браузерах с приемлемой производительностью (не более 30 с на одно изображение)
- Полный код решения занимает не более 10 МБ
